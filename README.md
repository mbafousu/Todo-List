# React Todo Learning Tracker

A modern Todo List application built with React and Vite demonstrating state management, component architecture, and performance-aware design

## Overview 

- This project showcases a fully interactive Todo system with:

- create, edit, toggle, and delete functionality
- Inline editing with validation

- Immutable state updates

- Derived state optimization using `useMemo`

- Component-based architecture

## Tech Stack

- React (Functional Components + Hooks)

- Vite

- JavaScript (ES6+)

- CSS
